A sleek, responsive, and lightweight appointment booking portal for **Waris Optical**. Built using vanilla web technologies to ensure maximum performance and cross-browser compatibility.

---

## 🎯 Project Overview
This project focuses on providing a frictionless user experience for patients. By using **Vanilla JavaScript**, the application handles form validation and UI transitions without the overhead of heavy libraries.

### ✨ Key Features
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop using CSS Flexbox/Grid.
* **Dynamic Form Handling:** Real-time feedback using JavaScript event listeners.
* **Custom Styling:** Bespoke CSS tailored to the optical clinic's professional branding.
* **Form Validation:** Client-side checks to ensure all patient data is captured correctly before submission.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for SEO and accessibility.
* **CSS3:** Custom properties (variables), animations, and responsive layouts.
* **JavaScript (ES6+):** DOM manipulation and appointment logic.
* **Hosting:** [Render](https://render.com/) (Static Site Deployment).
